# DDCS

DDCS，全称Docker Desktop Chinese Script，即Docker汉化脚本。

MAC版本请切换到MAC分支。

<big>**你可以在这个仓库找到各个版本的汉化包：【 https://github.com/asxez/DockerDesktop-CN 】**</big>

## 环境需求
- python3.10+
- nodejs

## 使用方法
下载源码，管理员权限启动终端并进入到源码根目录，使用以下命令即可：
```bash
python ddcs.py
```
注意：请务必使用管理员权限启动终端。
